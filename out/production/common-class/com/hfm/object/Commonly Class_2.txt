2.常用类
2.1.System类
	System系统类 ： 主要的作用是用于获取系统的一个参数。
	System类需要掌握的方法：
		arraycopy(Object src, int srcPos, Object dest, int destPos, int length):拷贝数组
			src 	- 源数组。
			srcPos 	- 源数组中的起始位置。
			dest 	- 目标数组。
			destPos - 目标数据中的起始位置。
			length 	- 要复制的数组元素的数量。
		currentTimeMillis() :获取当前的系统时间
		exit(int status) 	:退出jvm，0表示正常退出jvm，非0 表示异常退出。
		gc() 				:建议jvm尽快的启动垃圾回收器回收垃圾。
		getenv(String name)	:获取环境变量
		getProperties()		:获取系统属性
		getProperty(key)	:获取系统属性by key
2.2.Runtime类
	代表了当前程序的运行环境。
	maxMemory()		:JVM最大内存数
	totalMemory()	：总内存数
	freeMemory()	：当前可用内存
	exec(path)		：执行并返回一个线程
	Process(线程类)	：接收exec返回的线程
2.3.Date类(已经过期)
	Date 类封装的是系统的当前时间.。但是 Date 已经过时了，sun推荐使用 Calendar类。
	Calendar: 该类是一个日历的类，封装了年月日时分秒时区。
	SimpleDateFormat 日期格式化类作用：
		1. 可以将时间对象转成指定格式的字符串。 format();
		2. 可以把字符串转成日期对象。  parse(); 
	get(Calendar.YEAR)	 		年
	get(Calendar.MONTH)			月(从0开始)
	get(Calendar.DATE)			日
	get(Calendar.HOUR_OF_DAY)	时
	get(Calendar.MINUTE)		分
	get(Calendar.SECOND)		秒
2.4.Math类	
	常用的方法：
		abs(double a) 	绝对值
		ceil(double a) 	向上取整 
		floor(double a)	向下取整
		round(double a)	四舍五入
		random() 		取随机数
